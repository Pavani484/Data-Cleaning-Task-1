# Data Cleaning and Preprocessing - Task 1

## Objective

The objective of this task was to clean and preprocess a raw dataset by handling missing values, removing duplicates, standardizing column names, converting data types, and preparing the dataset for analysis.

## Dataset

Netflix Movies and TV Shows Dataset

## Tools Used

* Python
* Pandas
* Jupyter Notebook

## Data Cleaning Steps Performed

1. Loaded the dataset using Pandas.
2. Inspected dataset structure and data types.
3. Identified missing values.
4. Filled missing values in:

   * director
   * cast
   * country
   * rating
   * date_added
   * duration
5. Checked and removed duplicate records.
6. Standardized column names.
7. Converted date_added column to datetime format.
8. Removed extra spaces from text fields.
9. Verified data types and data quality.
10. Saved the cleaned dataset.

## Results

* Total Rows: 8807
* Total Columns: 12
* Missing Values Remaining: 0
* Duplicate Records Remaining: 0
